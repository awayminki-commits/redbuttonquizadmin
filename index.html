🖼 이미지 업로드 방법

⚠ 구글 드라이브 링크는 사용 불가
Google Drive는 보안 정책상 외부 사이트에서 이미지를 직접 불러오는 것을 차단합니다. 아래 무료 이미지 호스팅 서비스를 이용해주세요.
① Imgur (가장 간단, 추천)
  1. imgur.com 접속
  2. 우측 상단 New post 클릭
  3. 이미지 업로드
  4. 업로드된 이미지 우클릭 → 이미지 주소 복사
  5. 복사한 URL을 아래 입력란에 붙여넣기
    (예: https://i.imgur.com/abcXYZ.jpg)
② 기타 이미지 호스팅 서비스
  • postimages.org — 회원가입 없이 업로드
  • imgbb.com — 무료 이미지 호스팅
💡 직접 URL 확인 방법
이미지 URL은 .jpg .png .webp 등 이미지 확장자로 끝나야 합니다.
브라우저에서 URL만 열었을 때 이미지만 보이면 정상입니다.

🎮 퀴즈 사이트 URL

응시자에게 공유되는 퀴즈 전용 사이트 URL을 입력하세요.
Admin URL과 분리하여 보안을 유지합니다.

🔗 구글 시트 연동 (Apps Script URL)

문제 은행을 구글 시트와 동기화하고 퀴즈 결과를 저장하는 데 사용됩니다.
Apps Script 배포 후 발급된 웹 앱 URL을 입력하세요.

🔒 API 보안 토큰

Apps Script 코드 상단의 SECRET_TOKEN 값과 동일하게 입력하세요.
토큰이 일치해야 구글 시트와 통신할 수 있습니다.

저장된 코드를 기반으로 퀴즈 앱 관리자 접속 링크를 생성합니다.
이 링크를 퀴즈 앱 링크 입력란에 붙여넣으면 설정 화면으로 진입합니다.
